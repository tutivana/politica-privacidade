# Política de Privacidade
Política de privacidade do Quiz App

Esta Política de Privacidade descreve como o aplicativo Quiz App Master CH (mz.co.quizappmasterch) trata as informações dos usuários.

1. Coleta de Informações

O aplicativo Quiz App Master CH não coleta, armazena nem compartilha informações pessoais dos usuários.

Não solicitamos dados como nome, e-mail, número de telefone, localização ou qualquer outra informação que possa identificar o usuário.

2. Uso de Informações

Como não coletamos dados pessoais, nenhuma informação é utilizada para fins comerciais, publicitários ou analíticos.

3. Compartilhamento de Dados

O aplicativo não compartilha dados com terceiros.

4. Anúncios

O aplicativo exibe anúncios.

5. Segurança

Mesmo não coletando dados pessoais, nos preocupamos com a segurança do aplicativo e adotamos medidas razoáveis para garantir uma experiência segura aos usuários.

6. Público-alvo

O aplicativo é destinado a usuários com 13 anos de idade ou mais.

7. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente. Qualquer alteração será refletida nesta página.
